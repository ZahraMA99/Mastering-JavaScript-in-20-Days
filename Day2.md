    5. null  
- Objects (e.g document & friends).

<br/>`"aloha".indexOf("a")` returns 0. "It take the first appearnce of the charachter."
<br/>`"sofia".indexOf("z")` returns -1. "convention in jsðŸš©! if characters not exist in the string, `indexof` will return -1".
